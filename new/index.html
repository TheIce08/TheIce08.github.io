更新日志

Limpid 是一个由 Marice 维护与开发的简约风格的基于 Hexo 的博客，所有的功能都与博客息息相关。可以让我轻松的分享知识、看法和见解。这里记录了本博客日常维护和更新的信息。

博客框架、主题？

  • 本站是基于 Hexo 静态博客框架部署，在 Anzhiyu 的基础上二次创作。
  • 网站托管于 Github 代码仓库和 Vercel。
  • 基础业务图床由去不图床提供。
  • 域名服务由腾讯云 DNSPod 提供。
