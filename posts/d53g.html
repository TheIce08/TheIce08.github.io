博客一周年记

Marice2023-12-02

去年今日此门中

时光飞逝，不知不觉间，我的小博客也度过了它的第一个生日。从一个简单的想法、一个自由发声的渠道，到构建出还算可观的博客，自豪地，我实现了从零到一的突破！纵观我开博以来一年，也许对一个前端后端小白来说，举步维艰…… 但对新知的自驱力促使着我探究、学习。

“眼睛是心灵的窗户，理至易明，我所看到的，若是符合我所想的，便是我将要付诸心力去完成的”

这来自外界的 诱惑 让我踏上了成为一个 Blogger 的小径。

记得那是一个暑假，看到 知乎、CSDN 类的可供个人撰写文章的平台，在我的心中便产生了这样一个想法：
相比微信朋友圈、QQ 说说之类的这些平台，是具有很大的局限性的，我能否搭建一个完全属于我自己的网络程序呢？
非比往日的 “十分钟热度” ，对探索天生感兴趣的我，在各路教程的世界中穿梭，终于，对前端可谓一窍不通的我成功构建了当今 blog.marice.top 的大概模样。这也是我能够坚持下来，不断魔改、坚持更新博客甚至到后来去开发更多实例小程序、网络应用的动力吧。

一年多的时间，我亲眼见证了博客世界的群星璀璨 ——
生活类博主深耕自我对外辐射光明与温暖；技术类博主领跑高新领域进行探索与专精；知识类博主如泱泱春蚕丰富视界。
在这一年，我也在寻找撰写博客的意义和吸取持续更新的动力。幸运地，能够在撰博这条道路上遇见非常幽默、勤奋、聪明的小伙伴。

从博客撰写的意义上来看，从想要作为新一代的个性体现，到帮助他人、传播互介再到与一些志同道合的好友们披荆斩棘、乘风破浪地前行。 就好比和本站友情链接中灰常多的小伙伴共同进步：他们很多与我身处异地、生活在不同的职业领域、可算是天差地别的年龄，但是我们都可以为了一个共同的大方向去努力，为中文博客这条路上添砖加瓦，这真是泰裤辣！

千锤百炼出真知

本博客是我在各路开源大佬的臂膀上一砖一瓦自行魔改构建的，糖果屋的朋友们见证了本站的成长。我个人非常支持开源精神，不仅仅是因为许许多多开源的程序方便了我的二次开发，同时也为现代 IT 领域起了个好头，我可以非常大声地喊出：我开源，我自豪！所以很多朋友都喜欢去 F12 借鉴一下 CSS 代码和样式模板之类的。

回望一年的种种经历，他们有力地证明了在没有人帮助的情况下也能通过自己的努力达成自己想要的效果。花费一些时间或吃一顿小苦，但这也是必要的，咬咬牙也就过去了，风雨过后见彩虹在此体现的淋漓尽致！

现在回过头看来，从按着大佬们的注释改代码到学会自力更生写代码；从自给自足找资源到开动脑筋创资源。这难道不是 “千锤百炼出真知” 吗？

领异标新二月花

开启创造力的关键是什么？很多时候，你无法在互联网上找到复杂问题的答案，毕竟不是所有人的操作方式都是相同的，这意味着你需要吸收和转化来自互联网的内容来应用到自己的实际场景。你在这样一个过程中同样提升了自我。

相反，当今市面上的各种定制化 AI 提供的解决方案遏制了你的自由想法，这就是问题所在。众所周知，诸多 AI 大模型的内容来自互联网，而互联网中发布的自然是已经被发掘的内容，所谓 “新思路” 也许只是拼凑处理而成” 新信息 “，因此，我们更需要抓好实际应用需求，尽可能自行开发，才能取得意想不到的成功。

秉持着这样一种信念，我开始设计和制作自己的小程序。最开始使用 Python 命令行运行：基础的 turtle、简易的计算工具、稍复杂的爬虫；后来，接触了 C++：计算一些数学问题；接着，易语言 凭借它的简便和轻量赢得了我的青睐：自动输入系统、倒数日（桌面级的时间管理器）都是基于它开发的。

至于前端，Vue.js 是我最喜爱的开发语言，更高度的集成化，更轻便的代码方式对于小白来说非常易于学习。在博客魔改的历程中，接触 HTML 5、CSS 3 是不可避免的，日积月累的前端知识也对我后续的页面自定义开发有了很大的帮助。

长风破浪会有时

我的开始

2022.10.29，十一月的开始，十月的结束。也许在某个网页上，也许在某个短视频内，我了解到了静态博客。不同以往 Wordpress、Halo 等项目，Hexo 这类 Severless 静态部署的网络程序让我大吃一惊。随便找了个 CSDN 上的 Hexo 静态部署教程，我来到了梦开始的地方：

紧跟教程，我注册了我的第一个 Github 账号，借此，我也被 Github 上不计其数的开源程序所震惊。

至于第一版博客的主题，我选择了 Butterfly，非常美丽的界面吸引了我的注意。第一次接触 ymal 格式 的配置文件，我深受其严格的缩进要求之苦：各种报错，各种编译失败…… 当然，最后都在互联网上得到了相应的解决办法。在这期间，我印象最深的博主是 “心流”，他转述、分类、增补的 Butterfly 主题文档对网络环境并不优良、对插件不甚知之的我提供了极大的便利：

详细的插件调用方案和介绍如导航般带我渐入佳境。

在这个阶段，我开始向 Hexo-Butterfly 纵深发展，插件、脚本魔改、前端样式优化成了那个时期的主要任务。其中，我接触到了糖果屋这个中文博客组织，因此，我印象最深的博主是 Akilar，他富有现代风格、与时俱进的博客让我眼前一新！

幸运地，我受到了糖果屋成员的热情接待，在店长和许许多多小伙伴们的帮助下，我完善了博客，学会了魔改并且收获了不少志同道合的朋友。

同样，这一过程也不是一帆风顺的，例如在将博客迁移至 Vercel 的时候，对自动化机制的不了解，使我在部署 GithubAction 时连连受挫…… 糖果屋的朋友们是非常热心的，其中 Decheng 大佬提供了无偿的远程协助（真的特别感动），但可惜的是，这位大佬在不久的更名升级后，我便无法联系到他了……

有了 GithubAction 的加持，撰写博文变得更加容易了。

蓦然回首，感觉当时那个小毛孩傻的可爱。现在看来易如指尖轻弹的部署，对当时的我来说简直难如登天……

我的创新

作为一个个人开发者，我是如何分配我的时间，达到学习开发两不误的呢？

我主要参与开发的平台为：西瓜创客（Scratch）、易语言（C#）、网络程序（HTML、CSS），分析经验，开发设计所占时间和学习时间几乎一致，而日常学业所占时间则为这些时间的 190%，约等于两倍，因此，留给我开发的时间是非常少的，那么精简需求、轻量化则为优化的重中之重。程序开发使用大量的既有模块其实不是一个好习惯，但是对流光似锦的我来说，这便是最优的选择，各个大佬封装好的模块为我的开发打下了坚实的基础。

容易得出，细致的规划和充足的准备是对时间充分利用的关键。虽然灵感往往来自于不经意的某个瞬间，但是给自己生成一个大纲还是利大于弊的。

我的近况

临近中考，学业上的负担不觉重了起来，很多不大重要的事情都从日程中悄然离去。博客更新升级之后，我的日浏览量（PV）最高时可达到 400+，对于一个资源不算丰富、内容不算充实的小站来说，这是令我意外欣喜的！

很高兴全国各地的朋友们通过各种渠道来到我的网站！也期待共同进步，共创辉煌！

感恩相伴 一路有你
